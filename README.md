# Radio

**Live app: [https://lunarem.com/radio/](https://lunarem.com/radio/)**


A simple web radio player built with React. Browse and stream Polish radio stations directly in the browser — no account or installation required.

## Features

- Stream live radio stations with a single click
- Play/pause controls and volume adjustment
- Persistent player bar while browsing stations
- Responsive layout — works on desktop and mobile

## Stations

- **Radio 357**
- **Radio Nowy Swiat**

## Tech Stack

- React 18
- Vite 6
- Bootstrap 5

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
